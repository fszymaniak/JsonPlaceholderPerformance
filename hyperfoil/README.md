# Hyperfoil Performance Tests

Microservices-oriented distributed load testing tool written in Java.

## Installation
Download from https://hyperfoil.io/download

## Features
- Distributed testing
- Microservices focus
- Kubernetes native
- Complex scenarios
- Real-time stats

## Running Tests
```bash
# Start controller
bin/controller.sh

# CLI
bin/cli.sh

# Run benchmark
run smoke-test.hf.yaml
```

## Configuration
YAML-based with phases, scenarios, and sequences.
